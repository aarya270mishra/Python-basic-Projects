# 🐍 Python Beginner Projects

Welcome to my **Python Beginner Projects** repository! 🚀

This repository contains a collection of beginner-friendly Python projects that I built while learning Python fundamentals. These projects helped me strengthen my understanding of variables, loops, functions, dictionaries, conditionals, file handling, and user interaction.

---

## 📂 Projects Included

### ⏰ 1. Alarm Clock
**File:** `Alarm_clock_pp.py`

A simple alarm clock application that allows users to set a timer and plays an alarm sound once the specified time is reached.

**Concepts Used**
- Time module
- Loops
- Functions
- Audio playback
- User input

## 🚀 How to Run

1. Clone this repository

```bash
git clone https://github.com/aarya270mishra/python-beginner-projects.git
```

2. Navigate to the project folder

```bash
cd python-beginner-projects
```

3. Run any project

```bash
python filename.py
```

Example:

```bash
python miniorder.py
```
